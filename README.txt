# Friday night funkin' Test making document
Okay so let me get to the point, This is a document demonstrating Every step and requirement you gotta have to be able to make a Friday night funkin test. It may or may not seem complex but just bare with me.


## First Step:
Go to [Unity.com](https://unity3d.com/get-unity/download) and Download the unity Hub.

Once downloaded, install then open it and go to the Installs Tab, Click Add, Choose Unity 2019, click next and don't choose a thing on the modules list, then click Done.

While it's downloading, go to https://code.visualstudio.com/download and download visual studio code.

Once Everything is done downloading, go to the Projects Tab and click new; name your project whatever, but make sure you choose 2D.
